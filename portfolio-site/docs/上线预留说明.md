# 上线结构说明

已注册域名：`yaoliumu.me`

## 公开展示结构

```text
yaoliumu.me
└─ www.yaoliumu.me
   ├─ /
   │  个人作品集首页
   ├─ /resume
   │  网页版简历，不提供 PDF 下载
   ├─ /pharmacy-system
   │  乡镇药店管理系统项目展示页
   └─ /music-player
      NewPlayer 轻量级本地音乐播放器展示页，下载跳转到 Gitee Release

admin.yaoliumu.me
└─ /
   药店管理系统管理端前端

client.yaoliumu.me
└─ /
   药店管理系统用户端前端
```

## 不公开展示

- `www.yaoliumu.me/downloads`：不提供该下载目录。
- 简历 PDF 下载：不提供，简历只保留网页版。
- NewPlayer 安装包：不放在本站下载，统一跳转到 Gitee Release。
- `www.yaoliumu.me/api`：可以作为 Nginx 内部反向代理路径存在，但不在作品集页面、项目展示页或简历页公开展示。

## 当前页面要求

- 首页只展示作品集内容、药店管理系统管理端入口、用户端入口和 NewPlayer Gitee 下载入口。
- 简历页不提供下载按钮。
- 药店管理系统展示页不提供下载地址。
- 页面不公开后端接口地址、服务器内部路径、数据库信息。
